⚠️ 使用说明：
1. 手机和电脑连接同一 WiFi
2. 用 Chrome 浏览器打开此页面
3. 点击"开始采集"并授权传感器
4. 保持手机屏幕常亮

🔧 如果显示"无数据"：
Chrome 在 HTTP 下会禁用传感器。解决方法：
1. 地址栏输入 chrome://flags
2. 搜索 unsafely-treat-insecure-origin-as-secure
3. 填入 http://192.168.10.131:5000
4. 设为 Enabled → 点底部 Relaunch
5. 重新打开此页面即可

💡 iOS注意：需 iOS 12.2+，Safari 可直接用
